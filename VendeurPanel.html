📜 Règlement du Marketplace Sibylla

Bienvenue sur le Marketplace Sibylla !

En utilisant ce marketplace, vous acceptez de respecter les règles suivantes :

🔹 Règles générales

  • Seuls les membres de Sibylla peuvent utiliser ce marketplace
  • Les transactions se font en aUEC (monnaie virtuelle in-game uniquement)
  • Soyez honnête dans vos descriptions d'items
  • Respectez les autres vendeurs et acheteurs

🔹 Transactions

  • Les transactions se font de gré à gré entre joueurs
  • Contactez le vendeur sur Discord pour finaliser la vente
  • Mettez à jour le statut de vos annonces (Disponible / Réservé / Vendu)
  • Supprimez vos annonces une fois l'item vendu

🔹 Interdictions

  • ❌ Pas de vraie monnaie (€, $, etc.) — Uniquement aUEC
  • ❌ Pas de spam ou d'annonces abusives
  • ❌ Pas d'arnaques ou de fausses annonces
  • ❌ Pas de vente de vaisseaux (trading in-game uniquement)
  • ❌ Pas de contenu à caractère sexuel

🔹 Sanctions

  • En cas de non-respect, votre compte vendeur peut être suspendu
  • Les règles appliquées sur le serveur Discord sont les mêmes pour le site
  • Tout litige peut faire l'objet d'un ticket modérateur

⚠️ Ce marketplace est un service gratuit fourni par Sibylla. Utilisez-le de manière responsable !
